# 🚀 Mars Jump Mission

An interactive learning experience for kids to understand why humans jump higher on Mars.

## ✨ Features
- Interactive storytelling
- Gravity comparison
- Gamified learning
- Personalized experience
- Responsive UI

## 🛠 Tech Stack
- React + TypeScript
- Tailwind CSS
- Framer Motion
